Category Analytics Dashboard

Описание:
Этот проект предназначен для анализа товарных категорий и управления запасами. Он помогает принимать решения по закупкам и оптимизации ассортимента с использованием ABC/XYZ-анализов и автоматических расчетов показателей эффективности.

Основной функционал:
✅ ABC-анализ — приоритизация товаров по вкладу в выручку (правило 80/20)
✅ XYZ-анализ — классификация товаров по стабильности спроса (коэффициент вариации)
✅ Управление запасами — автоматический расчет дней покрытия и оборачиваемости
✅ Рекомендации по заказам — светофорная система: срочно / план / норма, с расчетом бюджета закупки

Использование:
Загрузите Excel-файл с данными по продажам и остаткам товаров.
Используйте дашборд для анализа ABC/XYZ и управления запасами.
Получите рекомендации по закупкам на основе автоматических расчетов.

Технологии:
Excel 

Цель проекта:
Помочь менеджерам по закупкам и аналитикам быстро принимать обоснованные решения по ассортименту и запасам, экономя время и снижая риск излишков или дефицита.
